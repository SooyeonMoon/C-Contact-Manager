# C언어 전화번호부 프로젝트 (Phone Book_linked4)
이번 단계에서는 프로그램이 종료되어도 데이터가 사라지지 않게 파일입출력 기능 구현

### 핵심 포인트
1. **출력의 차이**
   - printf는 모니터(화면)에 출력
   - fprintf는 파일에 출력, 즉 파일에 글자를 적는 것
  
2. **입력의 차이**
   - scanf는 키보드로 입력
   - fscanf는 파일에 적힌 내용을 읽어옴
  
3. **데이터 저장**
   - 이전에는 프로그램 종료 시 RAM에 저장된 데이터가 모두 사라졌지만, 파일입출력 기능을 구현하여 하드디스크에 기록되므로 프로그램을 종료해도 데이터가 사라지지 않고 남아 있다.

### 노드 저장 방식
- addPerson : contact.txt를 'wt' 전용으로 열어 데이터를 기록
- loadFromFile : contact.txt를 'rt' 전용으로 열어 main에서 프로그램 시작 시 파일에 적힌 데이터를 읽어옴
